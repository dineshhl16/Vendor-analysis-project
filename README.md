# 📊 Vendor Performance Analysis

## 📌 Project Overview
This project focuses on analyzing vendor performance, sales, inventory, and profitability data to generate actionable business insights. Using Python, SQL, Excel, and Power BI, the project evaluates vendor contributions, identifies profitable products, and supports data-driven inventory and procurement decisions.

## 🎯 Objective
To analyze sales, inventory, and vendor data to improve profitability, optimize inventory management, and support strategic vendor selection.

## 🛠️ Technologies Used
- Python
- SQL
- Excel
- Power BI
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Workflow
1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. SQL-Based Data Analysis
5. Vendor Performance Evaluation
6. Inventory Analysis
7. Profitability Analysis
8. Dashboard Development
9. Business Insights Generation

## 🔍 Key Analysis Performed
- Vendor-wise sales and profit analysis
- Brand performance evaluation
- Inventory turnover analysis
- Purchase and sales trend analysis
- Profit margin assessment
- Stock optimization insights

## 📈 Results
- Analyzed 10,000+ sales, inventory, and vendor records
- Evaluated 100+ vendors and brands
- Identified high-performing vendors and profitable products
- Generated insights for pricing, procurement, and inventory management
- Built interactive Power BI dashboards for business decision-making

## 🚀 How to Run

### Clone the Repository
```bash
git clone https://github.com/yourusername/vendor-performance-analysis.git
```

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook
```bash
jupyter notebook
```

### Run the Notebook
Open the notebook file and execute all cells sequentially.
